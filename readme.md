# Hotel Reservation Application (4 Participants)

Tools and Languages
- Python

Frontend
- Python Tkinter and Tkinter TTK Bootstrap Library
- Python Pillow Library
- Python Requests Library

Backend
- Python FastAPI and FastAPI APIRouter Library

**Main Requirement**
- Implement Object Oriented Programming Method and SOLID principle
- User can signup with personal data and login with email and password
- User can search hotel by country and province and view hotel name and room type from result 
- User can search any visited reservation of registered customer
- Customer can view personal data and all incoming and canceled reservations (require login)
- Customer can make hotel reservation by select hotel and room type in hotel search result (require login)
- Customer can make payment of reservation by select payment method, input payment datas, select to use coupon or not, and confirm or cancel payment (require login)
- Customer can cancel incoming reservation by input reservation id (require login)
- Customer can add and delete favourite hotel to list in hotel search result (require login)
- Admin can add or delete hotel from application (require login)
- Admin can cancel any customer reservation by input reservation id (require login)

Responsibility - Frontend Developer
- Work with other 1 participant in frontend team
- Use and arrange Tkinter and Tkinter TTK Bootstrap widgets to create user interface
- Use Pillow Library for hotel room type images 
- Use Requests Library for HTTP request and response methods to connect with backend
- Enable and disable button by clicking step to prevent payment and lacking data 

Project Clone Guides
- Open other Terminal and run cd to path of entire project
- Run pip install -r requirements.txt for installing libraries and dependencies
- Run python main.py (backend)
- Open Terminal and run cd to path of entire project
- Run python main.py (frontend)
